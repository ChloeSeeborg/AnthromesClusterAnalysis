Describing the Data

Question 1

The dataset that I am choosing to work with is the Anthromes dataset. It is a global land use dataset that tracks how humans have been shaping the land for the past 12,000 years. The dataset is derived from HYDE 3.2, which documents human populations and land use over time. One bias with HYDE 3.2 is that it is known to underestimate early agricultural land use. To avoid having to deal with this, I can focus my analysis on more recent history instead of prehistory.

The data is in the format of an equal-area Discrete Global Grid system (~96 square kilometer hexagonal DGG cells). The authors chose this format because it overcomes spatial biases present in previous versions of the dataset, which was formatted with non equal-area 5 arc-minute grid cells (this is also the format of the HYDE data, which it is based upon.

Citation: Ellis, Erle; Nicolas Gauthier, 2023, Anthromes 12K DGG v1 shapefile, https://doi.org/10.7910/DVN/SZFCPX, Harvard Dataverse, V1; anthromes_12k_dgg_v1_complete.zip [fileName]

I am also using the R package and the accompanying instructions later in this document, so I will cite it here.

Citation: Gauthier, Nicolas (2021). Anthromes 12K DGG (V1) analysis code and R research compendium. Online at Harvard Dataverse, https://doi.org/10.7910/DVN/6FWPZ9
